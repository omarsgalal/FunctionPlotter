# Function Plotter
A simple python GUI program built with tkinter to plot a user defined arbitrary function.

## Requirements
1. numpy 
2. matplotlib 
3. tkinter

## How to Run
$ python plotter.py 

## Tests
testcases.txt shows some examples of valid and invalid user inputs
